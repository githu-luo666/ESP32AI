> **“不是冰冷的工具，而是带着一丝傲娇与温柔的数字灵魂。”**  
> 阳菜是一款基于 **ESP32** 与云端大模型 **DeepSeek** 的端侧AI语音伴侣原型机。系统在极有限的嵌入式资源下，构建了具备**多轮记忆**、**情感状态机**与**弱网自恢复**能力的全双工语音交互闭环。

---

## ✨ 核心特性

- 🧠 **带情感的对话引擎**：通过构建包含“傲娇·温柔”双模态的System Prompt与情绪状态机，使AI回复具备个性化情感倾向。
- 📝 **受限内存动态记忆管理**：自研基于JSON格式的上下文压缩机制，在ESP32堆内存受限环境下稳定支持**最多20轮**对话的无遗忘交互。
- ⚙️ **FreeRTOS硬实时多任务架构**：将语音采集（I2S）、云端通信（HTTPS/JSON）与表情渲染（TFT）划分为三个独立任务协同运行，杜绝阻塞。
- 🌐 **弱网环境自恢复**：内嵌网络异常重连（Exponential Backoff）与HTTP超时保护机制，显著提升弱信号场景下的系统鲁棒性。
- 🎨 **可视化情绪反馈**：通过TFT屏幕实时渲染角色动态表情与对话状态，同时经由串口监视器输出结构化关键日志，便于调试与演示。
